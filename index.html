Web-scraping

I. What is Web scraping? Web scraping is the process of extracting/collecting useful data from the web in a structured manner. The scraping is usually done manually by a user, and the term refers to the automated process implemented using a web crawler/bot. Specific data is then gathered and copied into a spreadsheet/local database from the web. Web scraping is used by people and businesses who want to make use of the vast amount of publicly available data online to make smarter decisions.
